AllergyIntolerance

    • Key:  = Must support,  = QICore-defined extension,  = Is-Modifier

       Field Card. Type and Description
        attestedDate 0..1 DateTime
      Indicates the most recent date on which the recorder has asserted that the information represented by this AllergyIntolerance record is accurate.
        category 0..1 code
      Category of the identified substance.

      Binding: AllergyIntoleranceCategory (Required)
        code 0..1 Concept
      Code for an allergy or intolerance statement (either a positive or a negated/excluded statement). This may be a code for a substance or pharmaceutical product that is considered to be responsible for the adverse reaction risk (e.g., "Latex"), an allergy or intolerance condition (e.g., "Latex allergy"), or a negated/excluded code for a specific substance or class (e.g., "No latex allergy") or a general or categorical negated statement (e.g., "No known allergy", "No known drug allergies").

      Binding: value set (Preferred)
        criticality 0..1 code
      Estimate of the potential clinical harm, or seriousness, of the reaction to the identified substance.

      Binding: AllergyIntoleranceCriticality (Required)
        identifier 0..* List<Identifier>
      This records identifiers associated with this allergy/intolerance concern that are defined by business processes and/or used to refer to it when a direct URL reference to the resource itself is not appropriate (e.g. in CDA documents, or in written / printed documentation).
        lastOccurrence 0..1 DateTime
      Represents the date and/or time of the last known occurrence of a reaction event.
        note 0..* List<Annotation>
      Additional narrative about the propensity for the Adverse Reaction, not captured in other fields.
        onset 0..1 DateTime
      Record of the date and/or time of the onset of the Allergy or Intolerance.
        patient 1..1 Patient
      The patient who has the allergy or intolerance.
        reaction 0..* List<Reaction>
      Details about each adverse reaction event linked to exposure to the identified substance.
        reasonRefuted 0..1 Concept
      A code capturing the explanation of why the allergy or intolerance has been refuted. Should be specified only if the status is refuted.

      Binding: value set (Example)
        recorder 0..1 Practitioner | Patient
      Individual who recorded the record and takes responsibility for its content.
        reporter 0..1 Patient | RelatedPerson | Practitioner
      The source of the information about the allergy that is recorded.
        resolutionAge 0..1 Age
      The estimated patient age at which the allergy or intolerance resolved. Should be specified only if the status is resolved.
        status 0..1 code
      Assertion about certainty associated with the propensity, or potential risk, of a reaction to the identified substance (including pharmaceutical product).

      Binding: AllergyIntoleranceStatus (Required)
        type 0..1 code
      Identification of the underlying physiological mechanism for the reaction risk.

      Binding: AllergyIntoleranceType (Required)
