PlanDefinition.ActionDefinition.RelatedAction

  • Relationship to another action

    A relationship to another action such as "before" or "30-60 minutes after start of".

    • Key:  = Must support,  = QICore-defined extension,  = Is-Modifier

       Field Card. Type and Description
        actionIdentifier 1..1 Identifier
      The unique identifier of the related action.
        anchor 0..1 code
      An optional indicator for how the relationship is anchored to the related action. For example "before the start" or "before the end" of the related action.

      Binding: PlanActionRelationshipAnchor (Required)
        id 0..1 String
      unique id for the element within a resource (for internal references). This may be any string value that does not contain spaces.
        offset[x] 0..1 Duration | interval<Quantity>
      A duration or range of durations to apply to the relationship. For example, 30-60 minutes before.
        relationship 1..1 code
      The relationship of this action to the related action.

      Binding: PlanActionRelationshipType (Required)
